Syntax:

	alter database set <option>

Valid options are:

 - [drop_threshold](./drop_threshold)
 - [expiration_log](./expiration_log)
 - [expiration_num](./expiration_num)
 - [list_limit](./list_limit)
 - [select_points_limit](./select_points_limit)
 - [set_timezone](./set_timezone)
